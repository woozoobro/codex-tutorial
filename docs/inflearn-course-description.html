OpenAI Codex 1시간 실전 입문

AI 에이전트가 막막했다면,
Codex로 한 번 끝까지 가봅니다.

아이디어 하나를 문서, 리서치, 이미지 목업, 메일 초안, Next.js 랜딩 페이지, Vercel 배포까지 이어가며 AI 에이전트의 실제 작업 흐름을 경험합니다.

이 강의는 왜 만들었나요?

AI 에이전트는 이미 꽤 멀리 왔지만, 처음 시작하는 사람에게는 여전히 문턱이 높습니다. CLI가 낯설고, 개발 환경 셋업에서 막히고, GitHub나 배포 같은 단어만 나와도 부담스럽게 느껴질 수 있습니다.

이 강의는 OpenAI Codex를 처음 접하는 분들이 AI 에이전트가 실제로 어떻게 일하는지를 빠르게 감 잡을 수 있도록 만든 실전 입문 강의입니다. 복잡한 이론보다 실제 화면에서 일이 진행되는 흐름을 먼저 보여드립니다.

COURSE FLOW

아이디어에서 배포까지, 한 번에 이어지는 흐름

01

채팅에서 아이디어를 꺼내기

가상의 커머스 브랜드를 예제로 잡고, Codex와 함께 아이데이션을 시작합니다.

02

AGENTS.md로 프로젝트 맥락 저장하기

새 스레드에서도 이어서 작업할 수 있도록 프로젝트 방향과 기준을 정리합니다.

03

웹 리서치와 브랜드 문서 만들기

시장 조사, 브랜드 전략, 비주얼 방향을 문서로 쌓아 다음 작업의 재료로 사용합니다.

04

이미지 목업과 메일 초안 만들기

문서 맥락을 바탕으로 제품 이미지 시안을 만들고, Gmail 초안까지 이어갑니다.

05

Next.js 랜딩 페이지와 Vercel 배포

Plan mode로 작업 계획을 세우고, 로컬 확인과 배포까지 한 흐름으로 진행합니다.

수강 후 얻을 수 있는 것

  • Codex가 ChatGPT와 무엇이 다른지 이해할 수 있습니다.
  • AI 에이전트에게 문서 작성, 리서치, 개발 작업을 맡기는 흐름을 볼 수 있습니다.
  • AGENTS.md, GitHub, 브라우저 확인, 배포가 하나의 작업 흐름으로 어떻게 이어지는지 알 수 있습니다.
  • AI 에이전트를 단순한 답변 도구가 아니라 결과물을 함께 만드는 작업 환경으로 이해할 수 있습니다.

이런 분들께 추천합니다

  • OpenAI Codex를 처음 써보는 분
  • AI 에이전트가 실제로 무엇을 해주는지 궁금한 분
  • Claude Code나 CLI 기반 도구가 어렵게 느껴졌던 분
  • 문서, 리서치, 이미지, 웹사이트, 배포까지 한 흐름으로 경험해보고 싶은 분
  • ChatGPT에서 답변만 받는 수준을 넘어, 실제 결과물을 만들고 싶은 분

강의 진행 방식

이 강의는 이론을 길게 설명하기보다 실제 Codex 화면에서 작업이 어떻게 진행되는지 보여주는 데 집중합니다. 처음에는 채팅으로 아이디어를 정리하고, 이후 프로젝트 폴더 안에서 문서와 코드를 만들고, 브라우저로 확인한 뒤 배포까지 이어갑니다.

깊은 개발 지식은 필요하지 않습니다. 다만 강의 후반부에서 Next.js, GitHub, Vercel 같은 도구가 등장하므로 웹사이트가 만들어지고 배포되는 흐름을 처음 접하는 분들은 일부 용어가 낯설 수 있습니다.

OpenAI Codex로 아이디어 하나를 문서, 리서치, 이미지, 웹사이트, 배포까지 이어가며 AI 에이전트의 실전 작업 흐름을 경험하는 입문 강의입니다.
