# Codex Web (Cloud)

> 클라우드의 Codex에 위임

Codex는 OpenAI의 코딩 에이전트 — 코드 읽기·편집·실행. Codex 클라우드로 자체 클라우드 환경에서 백그라운드 (병렬 포함) 작업.

## 셋업

[Codex](https://chatgpt.com/codex)로 가서 GitHub 계정 연결. 리포 코드로 Codex가 작업 + 그 작업으로부터 PR 생성 가능.

Plus, Pro, Business, Edu, Enterprise 플랜에 Codex 포함. → [가격](../getting-started/pricing.md). 일부 Enterprise 워크스페이스는 Codex 액세스 전 [admin 셋업](../administration/enterprise-admin-setup.md) 필요할 수 있음.

## Codex Web 사용

### Prompting 학습

더 명확한 prompt, 제약 추가, 적합한 디테일 레벨 선택 → 더 나은 결과. → [Prompting](../concepts/prompting.md)
