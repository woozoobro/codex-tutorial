# YouTube Metadata - Codex 입문 1편

## 영상 포지션

- 대상: Codex를 처음 써보는 사람, AI 에이전트가 실제로 뭘 해주는지 감이 없는 사람
- 핵심 약속: 아이디어 하나를 실제 프로젝트 산출물로 이어가는 전체 흐름을 보여준다
- 예상 길이: 50-60분
- 톤: 입문자 친화적, 실전 데모 중심, 너무 개발자스럽지 않게

```text

```

1. OpenAI Codex 튜토리얼: AGENTS.md, GitHub, 이미지 생성, Next.js, Vercel까지

4. Codex로 1시간 안에 브랜드 프로젝트 만들기: 리서치, 이미지, 웹사이트, 배포까지
5. 처음 쓰는 Codex: 1시간 안에 프로젝트 하나 끝까지 만들기

## 제목 메모

- `97% 마스터하기`는 클릭감은 좋지만 과장처럼 보일 수 있다.
- `97% 이해하기`, `97% 감 잡기`, `97% 시작하기`가 입문 영상에는 더 안전하다.
- 현재 원픽은 `1시간 안에 Codex 97% 이해하기: 아이디어에서 배포까지`.
