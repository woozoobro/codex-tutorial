# Codex 앱 개요

> Codex command center

Codex 앱은 Codex 스레드를 병렬로 작업하는 집중 데스크톱 경험 — worktree 지원, 자동화, Git 기능 빌트인.

ChatGPT Plus, Pro, Business, Edu, Enterprise 플랜에 Codex 포함. → [가격](../getting-started/pricing.md)

## 시작하기

macOS와 Windows 가용. 대부분 기능 양 플랫폼 동일.

### 1. 다운로드 + 설치

- [macOS (Apple Silicon)](https://persistent.oaistatic.com/codex-app-prod/Codex.dmg)
- [macOS (Intel)](https://persistent.oaistatic.com/codex-app-prod/Codex-latest-x64.dmg)
- [Windows](https://get.microsoft.com/installer/download/9PLM9XGG6VKS?cid=website_cta_psi)
- [Linux 알림 신청](https://openai.com/form/codex-app/)

Intel Mac은 Intel 빌드 선택.

### 2. 열기 + 로그인

ChatGPT 계정 또는 OpenAI API 키. API 키 사용 시 일부 기능 (예: [클라우드 스레드](../concepts/prompting.md)) 가용 안 할 수 있음.

### 3. 프로젝트 선택

작업할 프로젝트 폴더 선택. 이전에 Codex 앱·CLI·IDE 확장 사용 적 있으면 과거 프로젝트 표시.

### 4. 첫 메시지

**Local** 선택 확인 (로컬 머신 작업) → 첫 메시지. 예:
- "Tell me about this project"
- "Build a classic Snake game in this repo."
- "Find and fix bugs in my codebase with minimal, high-confidence changes."

영감: [활용 사례](../getting-started/use-cases.md), 처음이면 [모범 사례](../resources/best-practices.md).

## Codex 앱 사용

### 프로젝트 간 멀티태스크

프로젝트 스레드를 나란히 실행 + 빠르게 전환.
