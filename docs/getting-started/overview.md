# 개요 (Overview)

## Codex: 코딩하는 모든 곳에서 동작하는 단일 에이전트

Codex는 OpenAI의 소프트웨어 개발용 코딩 에이전트다. ChatGPT Plus, Pro, Business, Edu, Enterprise 플랜에 포함되어 있다. Codex로 다음과 같은 작업을 할 수 있다:

- **코드 작성**: 만들고 싶은 것을 설명하면 Codex가 의도에 맞는 코드를 생성하며, 기존 프로젝트 구조와 컨벤션에 맞춰 동작한다.
- **낯선 코드베이스 이해**: 복잡하거나 레거시 코드를 읽고 설명하여, 팀이 시스템을 어떻게 구성했는지 파악할 수 있도록 돕는다.
- **코드 리뷰**: 코드를 분석해 잠재적 버그, 로직 오류, 처리되지 않은 엣지 케이스를 찾아낸다.
- **디버깅 및 문제 해결**: 문제가 발생했을 때 실패 추적, 근본 원인 진단, 타겟팅된 수정안 제안까지 도와준다.
- **개발 작업 자동화**: 리팩토링, 테스트, 마이그레이션, 셋업 같은 반복 워크플로를 Codex가 처리하므로, 더 높은 수준의 엔지니어링에 집중할 수 있다.
