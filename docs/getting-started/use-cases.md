# 활용 사례 (Use Cases)

> 원문 페이지는 갤러리 형태로 40개 이상의 활용 사례를 카테고리별로 보여준다. 최신 사례는 원문에서 확인하고, 이 문서는 카테고리 구조만 정리한다.

## 5대 컬렉션

| 컬렉션 | 설명 |
|--------|------|
| **Production systems** | 실제 코드베이스를 탐색하고, 통제된 변경을 가하며, 반복 작업을 코드화하고, 품질을 유지 |
| **Productivity & Collaboration** | 데이터 분석, 서비스 결합, 인사이트를 액션으로 변환 |
| **Web development** | 디자인 입력을 반응형 UI로 변환, 빠른 반복 사이클 |
| **Native development** | iOS·macOS 앱 빌드, 리팩토링, 검증 |
| **Game development** | 초기 프로토타입부터 프로덕션 품질까지 게임 개발 |

## 대표 워크플로

1. **GitHub 코드 리뷰 통합** — 사람 리뷰 전에 회귀(regression) 및 잠재 이슈를 잡는다
2. **반응형 프론트엔드 디자인** — 시각 레퍼런스를 시각 검증과 함께 반응형 UI로 변환

## 카테고리별 예시

- **iOS / macOS**: App Intents, 텔레메트리 추가, SwiftUI 리팩토링
- **데이터 분석 & 리포팅**: 데이터 파이프라인 분석, 리포트 생성
- **자동화**: 버그 트리아지, 이메일 관리, 작업 우선순위
- **통합 워크플로**: Slack, Linear, ChatGPT 연동
- **프론트엔드**: 디자인 → 구현
- **코드 유지보수**: 마이그레이션, 리팩토링

## 필터링

원문 페이지에서 카테고리, 팀, 작업 유형, 플랫폼별로 필터링하여 적합한 워크플로를 찾을 수 있다.
